# Домашнее задание: Инициализация системы (systemd)

## Окружение

Работа выполнена на виртуальной машине:

- Vagrant 2.4.9
- VirtualBox 7.2.6a
- Базовый образ: ubuntu/jammy64

---

## Цель работы

В рамках задания реализованы:

1. Systemd service + timer
2. Unit-файл для spawn-fcgi
3. Template unit для запуска нескольких экземпляров Nginx

---

## 1. Watchlog (service + timer)

Создан сервис, который каждые 30 секунд выполняет проверку файла  
`/var/log/watchlog.log` на наличие ключевого слова `ALERT`.

При обнаружении ключевого слова запись добавляется в системный журнал с использованием `logger`.

Запуск осуществляется через systemd timer.

Работа сервиса подтверждается записями в журнале systemd.

---

## 2. Spawn-FCGI

Создан unit-файл `spawn-fcgi.service`, запускающий `php-cgi` через `spawn-fcgi`.

Параметры запуска вынесены в конфигурационный файл  
`/etc/spawn-fcgi/fcgi.conf`.

После запуска сервиса создаётся Unix-сокет `/run/php-fcgi.sock`,  
процесс `php-cgi` работает под пользователем `www-data`.

---

## 3. Несколько инстансов Nginx

Создан шаблонный unit `nginx@.service`.

Реализован запуск двух независимых инстансов:

- nginx@first — порт 9001
- nginx@second — порт 9002

Каждый инстанс использует собственный конфигурационный файл и PID-файл.

Одновременная работа сервисов подтверждается:

- активным состоянием unit-файлов
- наличием PID-файлов в `/run`
- прослушиванием разных TCP-портов
- корректными HTTP-ответами с разных портов

---
