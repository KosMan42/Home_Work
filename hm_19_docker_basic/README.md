# Custom Nginx Docker Image

Кастомный Docker-образ nginx на базе alpine.
После запуска контейнер отдаёт изменённую стартовую страницу.

## Сборка образа
docker build -t my-nginx:1.0 .

## Запуск контейнера
docker run -d -p 8080:80 my-nginx:1.0

##Образ доступен в Docker Hub:
- https://hub.docker.com/r/kpfama/my-nginx

Ответ на вопрос:
образ - это шаблон файловой системы. из которого создают контейнер. Не выполняется сам по себе.
контейнер - это запущеный экземпляр образа. Может быть запущен/остановлен. Имеет свое состояни. Создается из образа.
Скомпилировать ядро в контейнере возможно, так как контейнер — это изолированный процесс и ему доступны инструменты сборки. Однако контейнер всегда использует ядро хоста, поэтому заменить/загрузить новое ядро «внутри контейнера» нельзя — для этого нужен хост или виртуальная машина.
